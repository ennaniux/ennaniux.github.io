2 Commutator of the second Fundamental form

We are using from now on the definition of curvature given by do Carmo:

\begin{equation} R_{ijk}^m\partial_m = \left\{\partial_{j}\Gamma_{ik}^m - \partial_{i}\Gamma_{jk}^m + \Gamma_{js}^m\Gamma_{ik}^s - \Gamma_{is}^m\Gamma_{jk}^s\right\}\partial_m \end{equation}

Hypersurfaces with normal \(N\), with \(\epsilon_1 = g(N,N)\)

\begin{equation} R_{ijkl} = \epsilon_1 \left( h_{ik}h_{jl} - h_{jk}h_{il} \right). \end{equation} \begin{equation} \nabla_k \nabla_l h_{ij} - \nabla_l \nabla_k h_{ij} = \sum_r R_{klir}h_{rj} + \sum_r R_{kljr}h_{ir}. \end{equation}

Then in codimension two with \(h = -\delta\)

\begin{equation} \nabla_j\nabla_j A_{kk} = \nabla_{k}\nabla_{k} A_{jj} + \epsilon_2 \left\{ A_{jj}A_{kr}^2 -A_{kk}A_{jr}^2 \right\} + \epsilon_1\left\{h_{jj}h_{kr}A_{kr} - h_{kk}h_{jr}A_{jr} + h_{jk}h_{kr}A_{rj} - h_{kj}h_{jr}A_{kr}\right\}, \end{equation}

and in our cases last two cancelled in any case since \(h = -\delta\) and \(A\) is diagonal:

\begin{equation} \nabla_j\nabla_j A_{kk} = \nabla_{k}\nabla_{k} A_{jj} + \epsilon_2 \left\{ A_{jj}A_{kr}^2 -A_{kk}A_{jr}^2 \right\} + \epsilon_1\left\{h_{jj}h_{kr}A_{kr} - h_{kk}h_{jr}A_{jr}\right\}, \end{equation}

2.1 Starshaped surfaces

  • In Euclidean spaces, \(g = dr^2 + r^2\sigma\).
    Parameterisation
    \(Y = u \xi\).
    Tangent basis
    \(Y_i = u_i \xi + u\xi_i = u_i\partial_r + \partial_i\).
    Normal vecor
    \(\hat{n} = \frac{-\varphi(u) \partial_r + \sigma^{kl}u_l\partial_k}{\sqrt{\varphi^2(u) + \varphi(u)|\tilde{\nabla} u|^2}}\)
    With ϕ = u2
    \(\hat{n} = \frac{-u^2 \partial_r + \sigma^{kl}u_l\partial_k}{\sqrt{u^4 + u^2|\tilde{\nabla} u|^2}}\)
    Tilt function
    \(\tau = \langle Y,\hat{n}\rangle\). This is the support function and more over, note \(Y = u\partial_r\).
  • In Elliptic space (sphere), Metric: \(g = dr^2 + \sin^2(r) \sigma\).
    Parameterisation
    \(Y = (u,\xi)\).
    Tangent basis
    \(Y_i = u_i \xi + u\xi_i = u_i\partial_r + \partial_i\).
    Normal vecor
    \(\hat{n} = \frac{-\varphi(u) \partial_r + \sigma^{kl}u_l\partial_k}{\sqrt{\varphi^2(u) + \varphi(u)|\tilde{\nabla} u|^2}}\)
    With ϕ = sin2(u)
    \(\hat{n} = \frac{-\sin^2(u) \partial_r + \sigma^{kl}u_l\partial_k}{\sqrt{\sin^4(u) + \sin^2(u)|\tilde{\nabla} u|^2}}\)
    Tilt function
    \(\tau = \langle E_0,\hat{n}\rangle\). Note: \(\sin(u)\partial_r = -E_0 + \cos(u)Y\), and \(\hat{n}\perp Y\).

The common feature is to define the tilt and height functions as

\begin{equation} \tau = \langle \hat{n}, \sqrt{\varphi}\partial_r \rangle = \langle \hat{n}, E_0 \rangle,\qquad \eta = \langle Y, E_0\rangle \end{equation}

it can be shown that:

\begin{equation} \nabla_j\tau = -g^{ir}A_{rj}\nabla_i\eta \end{equation} \begin{equation} \nabla_k\nabla_j\tau = -g^{ir}\nabla_kA_{rj}\nabla_i\eta - g^{ir}A_{rj}\nabla_k\nabla_i\eta, \end{equation}

Is in the second derivatives of the eight function that geometry plays a role:

\begin{equation} \nabla_k \nabla_i \eta = \epsilon_2 A_{ik} \tau + \epsilon_1 h_{ik}\eta. \end{equation}

In special cases when \(h = -\delta\): and using Codazzi

\begin{equation} \nabla_k\nabla_j\tau = -g^{ir}\nabla_rA_{kj}\nabla_i\eta - \epsilon_2 g^{ir}A_{rj} A_{ik} \tau + \epsilon_1 g^{ir}A_{rj} \delta_{ik}\eta. \end{equation}

Simplified version

\begin{equation} \nabla_j\nabla_j\tau = -\nabla_rA_{jj}\nabla_r\eta - \epsilon_2 A_{jj}^2 \tau + \epsilon_1 A_{jj} \eta. \end{equation}

2.2 Curvature estimates for surfaces in Euclidean spaces:

In this case we use the commutator formula with \(\epsilon_1 = 0\) and \(\epsilon_2 = 1\), and taking \(A\) diagonal and fix \(k=1\)

\begin{equation} \nabla_j\nabla_j A_{11} = \nabla_{1}\nabla_{1} A_{jj} + A_{jj}A_{11}^2 -A_{11}A_{jj}^2 \end{equation}

In Euclidean space: We use the support function (or starshapedness)

\begin{equation} \tau = \langle Y, \hat{n} \rangle \end{equation}, With \begin{equation} \nabla_j\nabla_j \tau = -\tau A_{jj}^2 \end{equation}

Using the test function:

\begin{equation} \phi = \log\left(\frac{A_{11}}{\tau} \right), \end{equation}

with first derivatives given by

\begin{equation} \nabla_j\phi = \frac{\tau}{A_{11}} \left( \frac{\nabla_j A_{11}}{\tau} - \frac{A_{11}\nabla_j \tau}{\tau^2} \right) = \frac{\nabla_j A_{11}}{A_{11}} - \frac{\nabla_j \tau}{\tau}, \end{equation}

For this case we have

\begin{equation} \nabla_{j}\nabla_{j} \phi = \frac{\nabla_{j}\nabla_{j}A_{11}}{A_{11}} - \left(\frac{\nabla_j A_{11}}{A_{11}}\right)^2 - \frac{\nabla_j\nabla_j\tau}{\tau} + \left(\frac{\nabla_j\tau}{\tau}\right)^2 \end{equation}

Then at the maximum of \(\phi\) and assuming \(A_{11} > 0\) we have:

\begin{equation} 0 \geq A_{11} \nabla_j\nabla_j \phi = \nabla_j\nabla_j A_{11} - \frac{A_{11}}{\tau}\nabla_j\nabla_j\tau = \nabla_j\nabla_j A_{11} + A_{11}A_{jj}^2 = \nabla_{1}\nabla_{1} A_{jj} + A_{jj}A_{11}^2 , \end{equation}

Sin \(F\) homogeneous degree one, \(F^{jj} A_{jj} = \psi\), then

\begin{equation} 0\geq F^{jj}\nabla_{1}\nabla_{1} A_{jj} + \psi A_{11}^2 \geq \nabla_1\nabla_1 \psi + \psi A_{11}^2, \end{equation}

And this gives a bound for \(|A_{11}|\).

3.5 Distribution and Frobenius theorem

A k-distribution on an n-dimensional manifold \(M\) is a choice of a k-dimensional vector space \(V_m \subset T_mM\) for all \(m\in M\).

A distribution is smooth if at each point of the manifold there is an open set \(U\) of \(M\) such that for every point in \(U\) there are k smooth vector fields in \(U\) which span \(V\).

A distribution is called involute if for any two vector fields lying in it, the lie bracket also lies in it.

A submanifold \(N\) of a manifold \(M\) is an integral manifold of a distribution \(V\) on \(M\) if for every \(p\in N\), \(d\psi(N_p) = V_{\psi(p)}\). (Maybe its clearer \(T_pN = V_p\)).

There exist an integral manifold of \(V\) through every point of \(M\) if and only if \(V\) is involute.
