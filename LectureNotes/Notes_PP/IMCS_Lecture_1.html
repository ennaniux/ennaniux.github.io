1 Review on numbers

We recall and set notation for the following set of numbers that we will be using through the course regularily

  • The set of Natural numbers \(\mathbb{N} = \{1, 2, ...,\}\). (Sometimes these are called whole numbers, and it may include \(0\)).
  • The set of Integers \(\mathbb{Z} = \{1, 2, ...,\}\).
  • The set of Rational numbers \(\mathbb{Q} = \left\{\frac{p}{q} : p,q \in \mathbb{Z}, q\neq 0\right\}\).
  • The set of Irrational numbers \(\mathbb{I}\) as those that can be represented in a line segment whose length is not rational.
  • The set of Real numbers \(\mathbb{R} = \mathbb{Q}\cup \mathbb{I}\).

2 Some interesting properties of integers

Here we briefly describe the following

  • On the set \(\mathbb{Z}\) it makes sense to talk about the sum (\(+\)), substraction (\(-\)) and product (\(\cdot\)).
  • In general the quotient of an integer is not an integer.
  • Notice that some integers can be expressed as the product of other integers.
  • Fundamental theorem of arithmetic.
  • Euclidean algorithm to find the greatest common divisor of two numbers.

3 Some properties of the real numbers

  • On the set \(\mathbb{R}\) it makes sense to talk about the sum (\(+\)), substraction (\(-\)), product (\(\cdot\)) and quotient (\(\div\))
  • A real number x is either a rational number or an irrational number. Is \(0\) a rational or an irrational number? Is \(\sqrt{2}\) rational or irrational?
  • Some real numbers have two decimal representations. Is it true that 0.9999999…… equals 1?
  • Decimal representation, Binary representation, Octal representation.

4 Propositional calculus and logic

In order to solve questions like: Is \(\sqrt{2}\) an irrational number? we need tools from logic. In mathematics we decide if a statement is true or false if it is possible to construct a sequence of logical statements leading to the resolution of the question.

  • A statement is anything we can say, write or otherwise express that is either true or false. Then we will assume that:
    1. Every statement is either true or false (bivalence)
    2. No statement is both true and false.
  • Examples
    • “The number 10 is even”
    • “The number 10 is smaller than the number 5”
    • “Numbers bigger than 20 are tasty”

5 Logical operators

  • Most sentences can be seen as a series of statements combined using connectives such as “and”, “or”, “if . . . then . . .”

For example the sentence “if it is raining and I go outside then I get wet” is constructed from the three simple statements:

  1. “It is raining.”
  2. “I go outside.”
  3. “I get wet.”

Whether the original sentence is true or not depends upon the truth or not of these three simple statements.

  • As a shorthand we shall use the letter T for true and F for false.
  • Although it is also valid to use 1 for true and 0 for false.

We will discuss five ways of forming new statements out of old ones, corresponding to the English expressions:

  • and
  • or;
  • not;
  • if, then;
  • if and only if

7 Example. "P only if Q"

What is this sentence saying? Can you construct the truth table?

  • It means that P is true only when Q true.
  • It mean that P cannot be true when Q is not true.
  • The statment "P only if Q" is false if \(P\) is true but \(Q\) is false.
  • It means that if \(P\) is false \(Q\) may be true or false.
\(P\) \(Q\) \(P\) only if \(Q\)
1 1 1

1 0 0

  • What about the sentence \(\sim Q \implies \sim P\) ?
  • What about \(\sim P \lor Q\)?
  • What about \(\sim (P \implies Q)\)?

We say that two symbolic statements are equivalent if they have the same truth table.

8 Tautology and Contradiction

  • A statement that is always true is called a tautology. Then its truth table has no value F.
  • A statement that is always false is called a contradiction. Then its truth table has no value T.

Let \(P\) and \(Q\) be two logical statements.

  1. We say that \(P\) implies \(Q\) if \(P \implies Q\) is a tautology.
  2. We say that \(P\) and \(Q\) are equivalent if \(P \iff Q\) is a tautology.

9 De Morgan laws

These are one of the most basic tautologies:

  1. \(\sim ( P \lor Q)\) is equivalent to \((\sim P) \land (\sim Q)\).
  2. \(\sim ( P \land Q)\) is equivalent to \((\sim P) \lor (\sim Q)\).

10 Arguments

Let \(P_1, P_2, \ldots, P_k\) and \(Q\) be logical sentences. Consider the statement

\begin{equation*} P_1 \land P_2\land \cdots \land P_k \implies Q. \end{equation*}
  1. The previous statement is called argument
  2. The statements \(P_1, P_2, \cdots, P_k\) are called the premises.
  3. The statement \(Q\) is called conclusion.
  4. An argument is valid if it is a tautology, that is, if the premises imply the conclusion (every line of the truth table is T). Otherwise it is called invalid.

11 Comments on the notation.

  • Sometimes the symbol \(\models\) is used to stress that the statement we are dealing with is a tautology, or that is valid.
  • Some authors use the symbol \(P \rightarrow Q\) for the statement "if \(P\) then \(Q\)", and they reserve the symbol \(\implies\) when the statement is an implication (tautology).
