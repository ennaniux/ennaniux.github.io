2 Review on numbers

We recall and set notation for the following set of numbers that we will be using through the course regularily

  • The set of Natural numbers \(\mathbb{N} = \{1, 2, ...,\}\). (Sometimes these are called whole numbers, and it may include \(0\)).
  • The set of Integers \(\mathbb{Z} = \{1, 2, ...,\}\).
  • The set of Rational numbers \(\mathbb{Q} = \left\{\frac{p}{q} : p,q \in \mathbb{Z}, q\neq 0\right\}\).
  • The set of Irrational numbers \(\mathbb{I}\) as those that can be represented in a line segment whose length is not rational.
  • The set of Real numbers \(\mathbb{R} = \mathbb{Q}\cup \mathbb{I}\).

3 Some interesting properties of integers

Here we briefly describe the following

  • On the set \(\mathbb{Z}\) it makes sense to talk about the sum (\(+\)), substraction (\(-\)) and product (\(\cdot\)).
  • In general the quotient of an integer is not an integer.
  • Notice that some integers can be expressed as the product of other integers.
  • Fundamental theorem of arithmetic.
  • Euclidean algorithm to find the greatest common divisor of two numbers.

4 Some properties of the real numbers

  • On the set \(\mathbb{R}\) it makes sense to talk about the sum (\(+\)), substraction (\(-\)), product (\(\cdot\)) and quotient (\(\div\))
  • A real number x is either a rational number or an irrational number. Is \(0\) a rational or an irrational number? Is \(\sqrt{2}\) rational or irrational?
  • Some real numbers have two decimal representations. Is it true that 0.9999999…… equals 1?
  • Decimal representation, Binary representation, Octal representation.

5 Compex Numbers

The set of complex numbers is denoted by \(\mathbb{C}\). On this set there are defined the operations of addition, substraction, multiplication and division. Every element \(z\in \mathbb{C}\) can be written as

\begin{equation} z = x + i y, \end{equation}

where \(x,y\in \mathbb{R}\) and \(i\) is a special complex number with the property that \(i^2 = -1\).

If \(z,w\in \mathbb{C}\) then we should be able to define the following concepts:

  • The addition/substraction \(z\pm w\).
  • The multiplication \(z\cdot w\) or simply \(zw\).
  • The real part \(\mbox{Re}(z)\).
  • The imaginary part \(\mbox{Im}(z)\).
  • The modulus of a complex number \(|z|\) and the argument (angle).
  • The conjugate of a complex number \(\overline{z}\).
  • Trigonometric representation of a complex number: \(z = re^{i\theta}\).
  • Moivre's formula. and the nth-root of a complex number \(z^{1/n}\).
  • Triangle inequality and domains in the complex plane.

6 Field

The set \(K\) is called a field if it satisfies the following axioms

  1. The two operations, sum (\(+\)) and product (\(\cdot\)) (also called multiplication) are well defined:
    • \(\forall x,y\in K\) there is a unique element \((x+y)\in K\).
    • \(\forall x,y\in K\) there is a unique element \((x\cdot y)\in K\).
  2. Addition is commutative. \(\forall x,y\in K : z + w = w + z\).
  3. Addition is associative. \(\forall x,y,z\in K : x + ( y + z) = (x + y) + z\).
  4. There is a unique element \(0\in K\) such that \(\forall x\in K: x + 0 = x\).
  5. Existences of additive inverse. \(\forall x\in K \, \exists ! (-x)\in K : x +(-x) = 0\).
  6. Multiplication is commutative. \(\forall x,y\in K : z \cdot w = w \cdot z\).
  7. Multiplication is associative. \(\forall x,y,z\in K : x \cdot ( y \cdot z) = (x \cdot y) \cdot z\).
  8. There is a unique element \(1\in K\) such that \(\forall x\in K: x \cdot 1 = x\).
  9. Existences of additive inverse. \(\forall x\in K \, \exists ! x^{-1}\in K : x \cdot x^{-1} = 0\).
  10. Multiplication distributes over addition. \(\forall x,y,z\in K : x \cdot (y + z) = x\cdot y + x \cdot z\).

7 Comments on notation

An intuitive definition of a set is enought to follow the content.

  • The sentence "\(a\) belongs to the set \(A\)" is written as \(a\in A\).
  • The sentence "for all the elements \(a\) in the set \(A\) the property \(P(a)\) holds" is written \(\forall a\in A : P(a)\).
  • The sentence "There is at least one element \(a\) in the set \(A\) such that the property \(P(a)\) holds" is written \(\exists a\in A : P(a)\).
  • The sentence "There is a unique element \(a\) in the set \(A\) such that the property \(P(a)\) holds" is written \(\exists ! a\in A : P(a)\).
